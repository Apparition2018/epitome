## 部署
1. 安装 Ecology9
2. weaver\Resin\setup.exe you → Install/Change → 服务 → 启动
    - Java Home
    - Service Name
    - Extra Params: -Dfile.encoding=GBK
3. 新建数据库 → 127.0.0.1:8085:/spa/smallApp/static4engine/engine.html#/system/CreateDB_key=8o49ut → 初始化数据
    - 验证码：weaver\ecology\WEB-INF\code
4. 重启 Resin，登录系统，sysadmin
5. 复制识别码 → 搜索：license 授权文件申请 → 提交 → 下载 license 文件 → 上传附件 → 提交
    - 客户名称、license 类型、license 用途
    - 产品类型、license 文件识别码、开始日期、结束日期、一级分部数（0）、license 文件用户数（0）、是否控制并发
6. 环境变量 classpath

| 类别       | 方法/属性                       | 作用说明                                                 |
|:---------|:----------------------------|:-----------------------------------------------------|
| 路由与组件替换  | rewriteRouteQueue           | 路由重写队列，用于替换整个页面，将系统原页面替换为自定义的 React 组件。              |
|          | overwritePropsFnQueueMapSet | 组件属性重写队列，用于修改现有组件的 Props（如按钮文字、表格列配置）。               |
|          | overwriteClassFnQueueMapSet | 组件类方法重写队列，用于覆盖组件实例的方法，修改组件行为逻辑。                      |
| API 接口拦截 | rewriteApiParamsQueueSet    | API 请求参数重写队列，用于在请求发送前修改接口参数。                         |
|          | rewriteApiDataQueueSet      | API 响应数据重写队列，用于在数据返回后修改接口响应内容。                       |
| 辅助工具方法   | getEcodeParams              | 获取环境参数，如当前用户信息、OA 版本、应用上下文等。                         |
|          | getAsyncCom                 | 异步加载组件，动态获取已发布的 ecode React 组件模块。                    |
|          | getCom                      | 获取组件配置，用于读取其他组件的配置信息。                                |
|          | setCom                      | 注册/发布组件，将自定义组件注册到全局，供其他模块通过 getCom 或 getAsyncCom 调用。 |
|          | checkPath / checkLPath      | 路径匹配检查，判断当前路由是否与指定路径匹配，用于路由拦截条件判断。                   |
|          | load                        | 动态加载应用，加载并执行一个已发布的 ecode 应用。                         |
|          | loadjs                      | 动态加载外部 JS 文件，在需要时按需加载第三方脚本或工具库。                      |
### 引入组件
1. React 方式
    ```
    import React from "react";

    import { Button } from 'antd';
    import { Wealnput } from 'ecCom';

    export default NewCom;
    import NewCom from './NewCom';
    ```
2. ecode 方式
    ```
    const { Button } = antd;
    const { Wealnput } = ecCom;

    ecodeSDK.exp(NewCom);
    ecodeSDK.imp(NewCom);
    ```

2. `window.weaJs.callApi({url: 'xxx', params: {}, medthod: 'GET'}).then(res => {}):`
### 其它参数获取
1. 获取登录人信息：`ecodeSDK.getEcodeParams(['ecode_params'])._user`
2. 获取当前登陆人的 userid
```
const { Is } = WeaTools;
const themeAccount = Is.getJSONObj('theme-account');
let userid = themeAccount ? themeAccount.userid : '';
```
