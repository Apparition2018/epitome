## 3.物态变化
```
# 温度
温度的定义：[物体的冷热程度]
温度计液体有[水银、酒精、煤油]
温度计使用方法：
    ①使用前应观察它的[量程、最小刻度值]
    ②使用时温度计玻璃玻璃泡要[全部浸入]被测液体中，不要碰到[容器低、容器壁]
    ③待温度计[示数稳定]后再读数
    ④读数时玻璃泡要[继续留在]被测液体中，视线与温度计中液柱[液面相平]
体温计的刻度范围通常为[35-42℃]
太阳表面约[6000]℃；水银的凝固点[-39]℃；酒精的凝固点是[-117]℃，金的熔点是[1064]℃

# 熔化和凝固
晶体物质的熔化特点是[固液共存、吸热、温度保持不变]
非晶体物质熔化特点是[没有固定的熔点]
白炽灯灯泡中的灯丝用钨制成，因为钨的[熔点高]，使用一段时间后，灯丝会变细，这是因为钨[升华]

# 液化和汽化
汽化的两种方法是[蒸发]和[沸腾]      
蒸发的定义：[液体在任何温度下都能发生的，并且只在液体表面发生的缓慢的汽化现象]
影响蒸发的因素：[液体温度]、[液面表面积]、[液面空气流动速度]
沸腾的定义：[在一定温度下，在液体内部和表面同时发生的剧烈的汽化现象]
液化的方法：[降低温度]和[压缩体积]
```

光在水中速度为真空中速度的[3/4]，在玻璃中速度为真空中速度的[2/3]

# 光的反射
[反射角]等于[入射角]
能从各个方向看到本身不发光的物体，是由于光射到物体上发生了[漫反射]的缘故
晚上，在桌面上铺一张白纸，把一小块平面镜平放在纸上，用手电筒光从镜子正上方垂直镜面直接照射，在侧面观察，会发现镜子比较[暗]，因为它发生了[镜面]反射；
    白纸比较[亮]，因为它发生了[漫]反射
   
# 平面镜成像
平面镜成像原理：[光通过镜面反射后发散，光线的延长线的交点形成像]
平面镜作用：[利用平面镜成像]、[改变光路]、[扩大视野空间]

# 光的折射
插入水中的筷子，在水中的部分看起来向[上]弯曲，装了水的桶底看起来变[浅]，都是由于光从[水]射向[空气]时发生[折射]的缘故，此时折射角[大于]入射角

# 光的色散
紫外线可以用来[验钞、杀菌]；红外线可以用来[遥控器、夜视仪、诊断]
五一佳节，在常州紫荆公园月季花展上，小明将红色滤色镜(即红色玻璃)挡在照相机镜头前给一株绿叶黄花的月季拍照，照片上该花卉的颜色是[黑叶黑花]
