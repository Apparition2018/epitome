# WMS
- Warehouse Management System
---
## 业务链路
```
┌───────────────────────────────────────────────┐
│                    入库                        │
│  采购入库 · 产成品入库 · 委外入库 · 杂项入库       │
│  销售退货入库 · 生产退料入库                      │
└──────────────────────┬────────────────────────┘
                       ↓
┌─────────────────────────────────────────────────────────┐
│                 库存管理                                 │
│  库存现有量(wm_material_stock)                           │
│  事务流水(wm_transaction) · 批次(wm_batch)                │
│  序列号(wm_sn) · 转移(wm_transfer) · 盘点(wm_stock_taking)│
└──────────────────────┬──────────────────────────────────┘
                       ↓
┌──────────────────────────────────────────────┐
│                    出库                       │
│  生产领料 · 销售出库 · 委外出库 · 杂项出库        │
│  物料消耗 · 采购退货出库                        │
└──────────────────────────────────────────────┘
```
