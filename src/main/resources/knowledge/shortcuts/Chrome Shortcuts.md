## 标签页和窗口
```
Ctrl + [                                    移动到块初始位置
Ctrl + N                                    打开新窗口
Ctrl + Shift + N                            打开新窗口（匿名模式）
Ctrl + T                                    打开并跳转到新标签页
Ctrl + Shift + T                            重新打开关闭的标签页（按照关闭顺序）
Ctrl + Tab | Ctrl + PgDn                    跳转到下一个打开的标签页
Ctrl + PgUp | Ctrl + Shift + Tab            跳转到上一个打开的标签页
Ctrl + 1~8                                  跳转到指定标签页
Ctrl + 9                                    跳转到最右标签页
Alt + Home                                  在当前标签页打开主页
Alt + ←                                     在当前标签页打开前一页（浏览历史）
Alt + →                                     在当前标签页打开后一页（浏览历史）
Ctrl + W | Ctrl + F4                        关闭当前标签页
Ctrl + Shift W | Alt + F4                   关闭当前窗口
Alt + Space, N                              最小化当前窗口
Alt + Space, X                              最大化当前窗口
```

## Google Chrome feature
```
Alt + F/E                                   打开 Chrome 菜单
Ctrl + Shift + B                            显示或隐藏书签栏
Ctrl + Shift + O                            打开书签管理器
Ctrl + H                                    在新标签页打开历史记录页面
Ctrl + J                                    在新标签页打开下载页面
Shift + E                                   打开 Chrome 任务管理器
Shift + Alt + T                             将焦点设置在 Chrome 工具栏第一项上
F10                                         将焦点设置在 Chrome 工具栏最右项上
F6                                          在标签栏/工具栏/书签栏切换焦点
Ctrl + G                                    跳到下一个匹配
Ctrl + Shift + G                            跳到上一个匹配
Ctrl + Shift + J | F12                      打开开发者工具
Ctrl + Shift + Delete                       打开“清理浏览数据”选项
F1                                          在新标签页打开 Chrome 帮助中心
F7                                          caret 浏览模式
Ctrl + F6                                   跳到页面内容
Alt + Shift + A                             关注到非活动对话框 ?
```

## 地址栏
```
Ctrl + Enter                                给站点名添加 www. 和 .com，并在当前页打开
Alt + Enter                                 打开新标签页并执行搜索
Alt + D | Ctrl + L | F6                     跳到地址栏
Ctrl + K/E                                  跳到地址栏并打开历史地址记录 
Shift + Delete                              删除地址记录
```
