# 数据类型
1. 数字类型 > 日期类型|二进制类型 > 字符类型
2. char > varchar，小于 50byte 建议使用 char
3. 精确数据使用 decimal，非精确数据使用 float

|数据类型|MySQL|SQL Server|Oracle|
|-------|:---:|:--------:|:----:|
