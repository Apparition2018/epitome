maven 生命周期：
    1.clean     清理项目
        pre-clean   执行清理前的工作
        clean       清理上一次构建生成的文件
        post-clean  执行清理后的文件
    2.default   构建项目（最核心）
        compile test package install
    3.site      生成项目站点
        pre-site    在生成项目站点前要完成的工作
        site        生成项目的站点文档
        post-site   在生成项目站点后要完成的工作
        site-deploy 发布生成的站点到服务器上
