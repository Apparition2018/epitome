# [FastGPT](https://doc.fastgpt.cn/docs/)
- 一个基于 LLM 大语言模型的知识库问答系统，提供开箱即用的数据处理、模型调用等能力。同时可以通过 Flow 可视化进行工作流编排，从而实现复杂的问答场景！
- RAG：Retrieval-Augmented Generation，检索增强生成
- [在线使用](https://tryfastgpt.ai)
