数据类型
数据类型 概述 示例
Boolean 原始类型 除 Object 以外的所有类型都是不可变的，
例如对字符串的操作一定返回了一个新字符串，
这些类型的值为“原始值”		 表示一个逻辑实体，可以有两个值：true 和 false
Null 一个字面量 全局对象
Undefined 一个没有被赋值的变量会有个默认值 undefined
Number 数字类型，JS 中只有一种数字类型：基于 IEEE 754 标准的双精度 64 位二进制格式的值 Number-demo
String 字符串类型，表示文本数据 String-demo
Symbol 符号类型，唯一的并且是不可修改的, 并且也可以用来作为 Object 的 key 的值 String-demo
Object 内存中的可以被标识符引用的一块区域，可以看作是一组属性的集合 Object-demo
Object 的两种属性
属性 特征 数据类型 描述 默认值
数据属性 [[Value]] 任何Javascript类型 包含这个属性的数据值。 undefined
[[Writable]] Boolean 如果该值为 false，则该属性的 [[Value]] 特性不能被改变。 true
[[Enumerable]] Boolean 如果该值为 true，则该属性可以用 for...in 循环来枚举。 true
[[Configurable]] Boolean 如果该值为 false，则该属性不能被删除，并且除了 [[Value]] 和 [[Writable]] 以外的特性都不能被改变。 true
访问器属性 [[Get]] 函数对象或者 undefined 该函数使用一个空的参数列表，能够在有权访问的情况下读取属性值。 undefined
[[Set]] 函数对象或者 undefined 该函数有一个参数，用来写入属性值。 undefined
[[Enumerable]] Boolean 如果该值为 true，则该属性可以用 for...in 循环来枚举。 true
[[Configurable]] Boolean 如果该值为 false，则该属性不能被删除，并且不能被转变成一个数据属性。 true
基于 IEEE 754 标准的双精度 64 位浮点数
