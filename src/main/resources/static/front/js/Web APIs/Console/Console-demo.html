属性
方法 概述 示例
log() 向 Web 控制台输出一条消息 console.log('this is a %s', 'log.')
info() 同 log()，Firefox 前面会有一个小图标 console.info('this is a %s', 'info.')
warn() 向 Web 控制台输出一条警告信息 console.warn('this is a %s', 'warn.')
error() 向 Web 控制台输出一条错误消息 console.error('this is a %s', 'error.')
group() 创建一个新的内联 group, 后续所有打印内容将会以子层级的形式展示 console.group("This is the outer level");
console.log("Level 2");
console.groupEnd();
groupCollapsed() 同 group()，区别在于 groupCollapsed() 打印出来的内容默认是折叠的
groupEnd() 闭合当前内联 group
time() 启动一个以入参作为特定名称的计时器 console.time("answer time");
alert("Click to continue");
console.timeLog("answer time");
alert("Do a bunch of other stuff...");
console.timeEnd("answer time");
timeEnd() 结束特定的 计时器 并以豪秒打印其从开始到结束所用的时间
timeLog() 打印特定 计时器 所运行的时间
trace() 输出一个 stack trace timeLog()
