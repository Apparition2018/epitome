方法
方法 概述 示例
Element.getAttribute() 返回元素节点的指定属性值，没有则返回 null 或 "" attr = element.getAttribute(attrName)
Element.getAttributeNode() 返回元素节点的指定 Attr 属性节点 attrNode = element.getAttributeNode(attrName)
Element.hasAttribute() 如果元素拥有指定属性，则返回true，否则返回 false element.hasAttribute(attrName)
Element.hasAttributes() 如果元素拥有属性，则返回 true，否则返回 false element.hasAttributes()
Element.removeAttribute() 从元素中移除指定属性 element.removeAttribute(attrName)
Element.removeAttributeNode() 移除指定的属性节点，并返回被移除的节点 removed = element.removeAttributeNode(attrNode)
Element.setAttribute() 把指定属性设置或更改为指定值 element.setAttribute(name, value)
Element.setAttributeNode() 设置或更改指定属性节点 replaced = element.setAttributeNode(attrNode)
Element.getBoundingClientRect() 返回元素的大小及其相对于视口的位置 domRect = element.getBoundingClientRect()
