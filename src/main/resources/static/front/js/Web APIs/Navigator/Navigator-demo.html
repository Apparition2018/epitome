Navigator
属性/方法 概述 示例
appCodeName 返回浏览器的代码名
所有浏览器均返回"Mozilla" 		navigator.appCodeName
appName 返回浏览器的名称 navigator.appName
appVersion 返回浏览器的平台和版本信息 navigator.appVersion
cookieEnabled 如果浏览器启用 cookie，则返回 true，否则返回 false navigator.cookieEnabled
onLine 如果浏览器正常联网，则返回 true，否则返回 false (Chrome, Safari)
如果浏览器脱机工作，则返回 true，否则返回 false (Firefox, IE) 		navigator.onLine
platform 返回运行浏览器的操作系统平台
返回值可能是："Win32", "Linux i686", "MacPPC", "MacIntel" 		navigator.platform
userAgent 返回浏览器的用户代理(user-agent)字符串 navigator.userAgent
