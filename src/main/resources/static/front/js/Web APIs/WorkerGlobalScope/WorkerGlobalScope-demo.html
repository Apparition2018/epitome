属性/方法 概述 示例
setInterval() 按照指定的周期来调用一个函数或执行一段代码 id1 = setInterval(func/code, delay)
clearInterval() 取消用 setInterval 设置的重复定时任务 clearInterval(id1)
setTimeout() 在指定的毫秒数后调用一个函数或执行一段代码 id2 = setTimeout(func/code, delay)
clearTimeout() 取消用 setTimeout 设置的定时器 clearTimeout(id2)
