GlobalEventHandlers
HTML DOM 事件对象
分类 属性 概述 支持标签
表单事件 onfocus 元素获得焦点
onblur 元素失去焦点
onfocusin 元素即将获取焦点，支持事件冒泡
onfocusout 元素即将失去焦点，支持事件冒泡
oninput 元素获取用户输入
onchange 用户改变域的内容 <input>, <select>, <textarea>
onsearch 用户向搜索域输入文本 <input type="search">
onselect 文本被选定 <input type="file/password/text">, <textarea>
onreset 重置按钮被点击
onsubmit 提交按钮被点击
鼠标事件 onclick 鼠标点击某个对象
ondblclick 鼠标双击某个对象
oncontextmenu 鼠标右键打开上下文菜单
onmouseenter 鼠标移到元素上
onmouseleave 鼠标移出元素外
onmouseover 鼠标移到元素上，支持冒泡事件
onmouseout 鼠标移出元素外，支持冒泡事件
onmousedown 鼠标按键按下
onmouseup 鼠标按键松开
onmousemove 鼠标移动
键盘事件 onkeydown 键盘按键按下
onkeypress 键盘按键按下并松开
onkeyup 键盘按键松开
框架/对象事件 onerror 当加载文档或图像时发生某个错误 <img>, <input type="image">, <object>, <script>, <style>
onload 某个页面或图像被完成加载
onunload 用户退出页面
onresize 窗口或框架被调整尺寸
onscroll 文档被滚动
