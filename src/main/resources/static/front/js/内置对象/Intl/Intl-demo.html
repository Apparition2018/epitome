Intl
属性/方法 概述 示例
Collator 用于语言敏感的字符串比较的对象
DateTimeFormat 用于语言敏感的日期和时间格式
DisplayNames 用于语言、地区和脚本显示名称的一致翻译
ListFormat 用于语言敏感的格式
NumberFormat 用于语言敏感的数字格式
PluralRules 用于多种敏感格式和特定语言
RelativeTimeFormat 用于语言敏感的相对时间格式
Segmenter 用于语言环境敏感的文本分割
getCanonicalLocales() 返回规范的语言环境名称
supportedValuesOf() 返回一个已排序的数组，其中包含实现支持的惟一日历、排序规则、货币、编号系统或单位值
