属性
属性 概述 示例
Number 在非构造器上下文中 (如：没有 new 操作符)，能被用来执行类型转换 Number([]); // 0
Number.EPSILON 表示 1 与 Number 可表示的大于 1 的最小的浮点数之间的差值 2.2204460492503130808472633361816E-16 或 2^-52
Number.MAX_SAFE_INTEGER 表示在 JavaScript 中最大的安全整数(2^53 - 1) 9007199254740991
Number.MIN_SAFE_INTEGER 表示在 JavaScript 中最小的安全整数(-(2^53 - 1)) -9007199254740991
Number.MAX_VALUE 表示在 JavaScript 里所能表示的最大数值，
大于 MAX_VALUE 的值代表 "Infinity"		 1.7976931348623157e+308
Number.MIN_VALUE 表示在 JavaScript 中所能表示的最小数值 5e-324
Number.NaN 表示“非数字”（Not-A-Number）。和 NaN 相同
Number.POSITIVE_INFINITY 正无穷大
Number.NEGATIVE_INFINITY 负无穷大
方法
方法 概述 示例
Number.isFinite() 用来检测传入的参数是否是一个有穷数 Number.isFinite(NaN); // false
Number.isFinite('0'); // false, 全局函数 isFinite('0') 会返回 true
Number.isInteger() 用来判断给定的参数是否为整数 Number.isInteger(Infinity); // false
Number.isInteger(true); // false
Number.isInteger([1]); // false
Number.isSafeInteger() 判断传入的参数值是否是一个 安全整数 Number.isSafeInteger(Math.pow(2, 53)); // false
Number.isSafeInteger(Math.pow(2, 53) - 1); // true
Number.isNaN() 确定传递的值是否为 NaN 和其类型是 Number Number.isNaN() & isNaN()
toExponential(fractionDigits) 以指数表示法返回该数值的字符串表示形式 77.2345.toExponential(4); // "7.7234e+1"
77.3456.toExponential(3); // "7.735e+1"
toFixed(digits) 使用定点表示法来格式化一个数 2.34.toFixed(1); // "2.3"
-2.34.toFixed(1); // -2.3 （由于操作符优先级，负数不会返回字符串）
(-2.34).toFixed(1); // "-2.3" （若用括号提高优先级，则返回字符串）
toPrecision(precision) 以指定的精度返回该数值的字符串表示形式 123.456.toPrecision(4); // "123.5"
123.456.toPrecision(8); // "123.45600"
123.456.toPrecision(2); // "1.2e+2"
toString([radix]) 回指定 Number 对象的字符串表示形式
radix 指定要用于数字到字符串的转换的基数(从2到36)		 (15).toString(16); // "f"
(2).toString(2); // "10"
valueOf() 返回一个被 Number 对象包装的原始值 var numObj = new Number(10);
typeof numObj; // object
var num = numObj.valueOf(); // 10
typeof num; // number
