正则表达式
断言（Assertions） ^, $, \b, \B, x(?=y), x(?!y), (?<=y)x, (?<!y)x
字符类（Character Classes） ., \d, \D, \w, \W, \s, \S, \t, \r, \n, \v, \f, [\b] ……
组和范围 x|y, (x), \n, (?<Name>x), (?:x)
量词（Quantifiers） x*, x+, x?, x{n,m} ...
Unicode 属性转义（Unicode Property Escapes）
exec():在一个指定字符串中执行一个搜索匹配。返回一个结果数组或 null。
RegExp 属性
对象 属性/索引 描述 示例
const re = /quick\s(brown).+?(jumps)/ig;
const result = re.exec('The Quick Brown Fox Jumps Over The Lazy Dog');
返回值 [0] 匹配的全部字符串 result[0]; // Quick Brown Fox Jumps
[1], ...[ n] 括号中的分组捕获 result[1]; // Brown
result[2]; // Jumps
index 匹配到的字符位于原始字符串的基于0的索引值 result.index; // 4
input 原始字符串 result.input;
// The Quick Brown Fox Jumps Over The Lazy Dog
正则 lastIndex 下一次匹配开始的位置 re.lastIndex; // 25
ignoreCase 是否使用了"i"标记使正则匹配忽略大小写 re.ignoreCase; // true
global 是否使用了"g"标记来进行全局的匹配. re.global; // true
multiline 是否使用了"m"标记使正则工作在多行模式
（也就是，^ 和 $ 可以匹配字符串中每一行的开始和结束（行是由 \n 或 \r 分割的），
而不只是整个输入字符串的最开始和最末尾处。） 		re.multiline; // false
source 正则匹配的字符串 re.source; // quick\s(brown).+?(jumps)
RegExp.test() & String.search & RegExp.exec() & String.match()
方法 概述
regexObj.test(str) 查看正则表达式与指定的字符串是否匹配，返回 true 或 false
str.search(regexp) 成功返回正则表达式在字符串中首次匹配项的索引。否则返回-1
regexObj.exec(str) 如果正则表达式不包含 g 标志，
则 str.match() 会返回和 RegExp.exec() 相同的结果。
否则 match() 只返回所有匹配的子字符串
