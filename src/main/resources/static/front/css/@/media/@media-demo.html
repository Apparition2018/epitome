媒体类型
类型 概述
all 所有设备
print 打印机和打印预览
screen 电脑屏幕，平板电脑，智能手机等
speech 屏幕阅读器等发声设备
媒体特性
特性 概述 示例
width 可视宽度
height 可视高度
orientation 定义屏幕方向 @media (orientation: portrait) {...} // 纵向
@media (orientation: landscape) {...} // 横向
resolution 定义设备的分辨率 @media (min-resolution: 72dpi) {...}
@media (max-resolution: 300dpi) {...}
响应式布局
利用 @media 实现
media 属性
当可视宽度小于等于960px引入外联样式 <link rel="stylesheet" href="@media.css" media="only screen and (max-width: 960px)">
