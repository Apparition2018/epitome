指令 概述 示例
v-cloak 和 CSS [v-cloak] { display: none } 一起使用时，可以隐藏未编译的 Mustache 标签直到组件实例准备完毕 <div id="app" v-cloak>
v-once 只渲染元素和组件一次。随后的重新渲染，元素/组件及其所有的子节点将被视为静态内容并跳过。可用于优化更新性能。 <span v-once>This will never change: {{ msg }}</span>
v-show 根据表达式之真假值，切换元素的 display CSS property <p v-show="status === 1">show</p>
v-if
内置 <template> 元素上使用条件指令，最终渲染不包含该元素
用 key 管理可复用的元素
v-for
VUE列表渲染 FOR-IN和FOR-OF的区别
