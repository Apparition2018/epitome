事件
分类 事件 JS 概述
绑定事件处理器 .on() addEventListener(), attachEvent() 绑定一个或多个事件
.off() removeEventListener(), detachEvent() 移除一个事件
.one() 绑定一个或多个事件，每个元素只执行一次事件处理函数
.trigger() dispatchEvent(), fireEvent() 触发事件以及事件的默认行为（如表单提交）
.triggerHandler() 触发事件
表单事件 .focus() onfocus 元素获取焦点时触发
.blur() onblur 元素失去焦点时触发
.focusin() onfocusin 元素即将获取焦点时触发，支持事件冒泡
.focusout() onfocusout 元素即将失去焦点时触发，支持事件冒泡
.change() onchange 元素内容改变时触发
.select() onselect 用户选取文本时触发
.submit() onsubmit 表单提交时触发
鼠标事件 .click() onclick 单击事件
.dblclick() ondblclick 双击事件
.contextmenu() oncontextmenu 右键打开上下文菜单时触发
.toggle() 绑定两个或多个处理程序绑定到匹配的元素，用来执行在交替的点击
.hover() 将二个事件函数绑定到匹配元素上，分别当鼠标指针进入和离开元素时被执行
.mouseenter() onmouseenter 鼠标移到元素上触发
.mouseleave() onmouseleave 鼠标移出元素外触发
.mouseover() onmouseover 鼠标移到元素上触发，支持冒泡事件
.mouseout() onmouseout 鼠标移出元素外触发，支持冒泡事件
.mousedown() onmousedown 鼠标按键按下时触发
.mouseup() onmouseup 鼠标按键松开时触发
.mousemove() onmouseve 鼠标移动时触发
键盘事件 .keydown() onkeydown 键盘按键按下时触发
.keypress() onkeypress 键盘按键按下并松开时触发
.keyup() onkeyup 键盘按键松开时触发
浏览器事件 .resize() onresize 当调整浏览器窗口大小时，触发 resize 事件
.scroll() onscroll 当用户滚动指定的元素时，触发 scroll 事件
文档加载 .ready() onload 当DOM准备就绪时，执行指定函数
$.holdRead() 暂停或恢复 .ready() 事件的执行
