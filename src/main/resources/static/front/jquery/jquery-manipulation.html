DOM 操作
分类 API JS 概述 例子/用法
通用属性 .attr() el.getAttribute(attrName)
el.setAttribute(name, val) 		设置或返回被选元素的属性/值 attr() 和 prop() 的区别
.prop() 设置或返回被选元素的属性/值
.removeAttr() el.removeAttribute(attrName) 从被选元素移除一个或多个属性 $(this).removeAttr('foo')
.removeProp() 移除通过 prop() 方法设置的属性 $(this).removeProp('foo')
.val() 设置或返回被选元素的属性值（表单） $(this).val()
class 属性 .addClass() el.classList.add(token1[, ...]]) 向被选元素添加一个或多个类名 $('p')
.removeClass('foo bar')
.addClass('baz')
.removeClass() el.classList.remove(token1[, ...]]) 从被选元素移除一个或多个类
.hasClass() el.classList.contains(token1[, ...]]) 检查被选元素是否包含指定的 class $('p').hasClass('foo')
.toggleClass() el.classList.toggle(token1[, ...]]) 在被选元素中添加/移除一个或多个类之间切换 $('p').toggleClass('foo')
CSS 属性 .css() el.currentStyle[attr]
getComputedStyle(el, null)[attr]
el.style.height = '...' 		为被选元素设置或返回一个或多个样式属性 $(this).css('backgroundColor')
$(this).css('color', 'red')
.height() 设置或返回被选元素的高度
.css('height') 和 .height() 之间的区别是：
.height() 返回一个没有单位的数值 		$(this).height(30)
.innerHeight() el.clientHeight 返回元素的高度（包含 padding，不包含 border） $(this).innerHeight()
.outerHeight() 返回元素的高度（包含 padding 和 border） $(this).outerHeight()
.width() 设置或返回被选元素的宽度 $(this).width(50)
.innerWidth() 返回元素的宽度（包含 padding，不包含 border） $(this).innerWidth()
.outerWidth() 返回元素的宽度（包含 padding 和 border） $(this).outerWidht()
.position() 返回元素相对于 offset parent 的坐标
offset parent 指离元素最近且被定位过的祖先元素 		$(this).position()
.scrollLeft() 设置或返回被选元素的水平滚动条位置 $(this).scrollLeft(300)
.scrollTop() 设置或返回被选元素的垂直滚动条位置 $(this).scrollTop()
.offset() 设置或返回被选元素的偏移坐标（相对于文档） $(this).offset()
.offsetParent() 返回第一个定位的父元素 $(this).offsetParent()
$.cssNumber 一个对象，这个对象包含所有可以不使用单位的CSS属性
用于.css()来确定是否需要附加px到无单位的值
DOM 插入并包裹现有内容 .wrap() 在每个被选元素外层包上一个html元素 wrap(), wrapAll(), wrapInner()
.wrapAll() 在所有被选元素外层包上一个html元素
.wrapInner() 在每个被选元素的内容外包上一个html元素
DOM 插入现有元素内 .html() el.innerHTML 设置或返回被选元素的内容 $(this).html('<p>Test<p>')
.text() el.textContent || el.innerText 设置或返回被选元素的文本内容 $(this).text(Test)
.prepend() pNode.prepend(Node or DOMStr) 在被选元素的开头插入内容 $('.foo').prepend('<p>Test<p>')
.prependTo() 将内容插入到被选元素的开头 $('<p>Test<p>').prependTo('.foo')
.append() pNode.append(Node or DOMStr)
node.appendChild(child) 		在被选元素的结尾插入内容 append() 和 after()的区别
.appendTo() 将内容插入到被选元素的结尾
DOM 插入现有元素外 .after() node.after(Node or DOMStr) 在被选元素后插入内容
.insertAfter() 将内容插入到被选元素后
.before() node.before(Node or DOMStr) 在被选元素前插入内容 $('.foo').before('<p>Test<p>')
.insertBefore() 将内容插入到被选元素前 $('<p>Test<p>').insertBefore('.foo')
DOM 替换 .replaceWith() node.replaceWith(Node or DOMStr) 把被选元素替换为新的内容 $('.foo').replaceWith('<p>Test<p>')
.replaceAll() 将新的内容替换被选元素 $('<p>Test<p>').replaceAll('.foo')
DOM 移除 .remove() node.remove() 移除被选元素（包含数据和事件） remove() 和 detach() 的区别
.detach() 移除被选元素（保留数据和事件）
.empty() el.innerHTML = '' 从被选元素移除所有子节点和内容 $(this).empty()
.unwrap() 移除被选元素的父元素 $(this)>unwrap()
DOM 复制 .clone() node.cloneNode(true) 创建被选元素的深拷贝副本 $('.foo').clone().appendTo('.bar')
