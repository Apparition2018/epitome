String
属性/方法 概述 示例
String 在非构造器上下文中 (如：没有 new 操作符)，能被用来执行类型转换 String(1)
length 表示一个字符串的长度 "abc".length
String.fromCharCode() 返回使用指定的Unicode值序列创建的字符串 String.fromCharCode(65,66,67)
charAt() 从一个字符串中返回指定的字符 "abc".charAt(1)
charCodeAt() 返回表示给定索引的字符的 Unicode 的值 "abc".charCodeAt(1)
concat() 将一个或多个字符串与原字符串连接合并，形成一个新的字符串并返回 ''.concat(1, 2, '3')
includes() 判断一个字符串是否包含在另一个字符串中，返回一个 boolean 值 'Hello World!'.includes('Hello')
startsWith()
endsWith()		 判断当前字符串是否是以另外一个给定的子字符串“开头/结尾”的，
返回一个 boolean 值		 'Hello World!'.endsWith('!')
indexOf() 返回调用 String 对象中第一次出现的指定值的索引 'Hello World!'.indexOf('World')
lastIndexOf()
localeCompare() 返回一个数字，来表示两个字符串的前后关系 'cat'.localeCompare('banana')
search() 对正则表达式和指定字符串进行匹配搜索，返回第一个出现的匹配项的下标 'banana'.search(/n/)
match() 使用正则表达式与字符串相比较 var str = 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz';
var regexp = /[A-E]/gi;
var matches_array = str.match(regexp);
// ['A', 'B', 'C', 'D', 'E', 'a', 'b', 'c', 'd', 'e']
replace() 返回一个由替换值替换一些或所有匹配的模式后的新字符串 replace()-demo
repeat() 返回指定重复次数的由元素组成的字符串对象 'abc'.repeat(3.5)
slice() 提取一个字符串的一部分，并返回一新的字符串 'I am hero'.slice(5)
substring() 返回一个字符串在开始索引到结束索引之间的一个子集,
或从开始索引直到字符串的末尾的一个子集		 'I am hero'.substring(5, )
substr() 返回一个字符串中从指定位置开始到指定字符数的字符 'I am hero'.substr(5, 4)
toUpperCase()
toLowerCase()		 将调用该方法的字符串值转换为大写/小写形式，并返回 'abc'.toUpperCase()
trim() 从一个字符串的两端删除空白字符 ' abc '.trim()
anchor() 创建一个 HTML 锚元素，被用作超文本靶标（hypertext target） 'A of Contents'.anchor('anchor_name')
link() 创建一个 HTML 元素，用该字符串作为超链接的显示文本，
参数作为指向另一个 URL 的超链接		 'baidu'.link('www.baidu.com')

可以在每行末尾使用反斜杠字符（“\”），
以指示字符串将在下一行继续
从字符串中获取单个字符
把字符串当作一个类似数组的对象，
其中的每个字符对应一个数值索引
