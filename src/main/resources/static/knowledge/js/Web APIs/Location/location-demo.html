属性/方法 概述 示例
https://developer.mozilla.org/en-US/search?q=URL#search-results-close-container
href 设置或返回完整的 URL href
origin origin
protocol 设置或返回当前 URL 的协议 protocol
host 设置或返回当前 URL 的主机名(域名)和端口号 host
hostname 设置或返回当前 URL 的主机名(域名) hostname
port 设置或返回当前 URL 的端口号 port
pathname 设置或返回当前 URL 的路径部分，开头有一个"/" pathname
search 设置或返回当前 URL 的查询部分，开头有一个"?" search
hash 设置或返回当前 URL 的片段标识符，开头有一个"# hash
assign() 加载新的文档，可以通过浏览器后退按钮返回到原页面 location.assign('http://baidu.com')
replace() 用新的文档替换当前文档，不可以通过浏览器后退按钮返回到原页面 location.replace('http://baidu.com')
reload() 重新加载当前文档
true: 强制浏览器从服务器加载页面资源
false: 浏览器则可能从缓存中读取页面 		location.reload(true)
