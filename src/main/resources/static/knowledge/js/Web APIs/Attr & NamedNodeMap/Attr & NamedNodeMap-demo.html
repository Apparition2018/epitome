Attr & NamedNodeMap
属性 概述 示例
Attr.name 返回属性的名称
Attr.value 设置或返回属性的值
NamedNodeMap.getNamedItem() 从 NamedNodeMap 返回指定的属性节点，或 null myAttr = attrs.getNamedItem(name)
NamedNodeMap.removeNamedItem() 移除指定的属性节点 removed = attrs.removeNamedItem(name)
NamedNodeMap.setNamedItem() 设置指定的属性节点 new = attrs.setNamedItem(name)
