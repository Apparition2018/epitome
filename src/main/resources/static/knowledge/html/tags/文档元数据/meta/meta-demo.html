meta
属性 概述 值 作用 示例
charset 定义当前文档所使用的字符编码 <meta charset="UTF-8" >
content 定义与 http-equiv 或 name 属性相关的元信息 <meta http-equiv="X-UA-Compatible" content="ie=edge,chrome=1">
http-equiv 把 content 属性关联到 HTTP 头部 default-style 页面首选样式表 <meta http-equiv="default-style" content="defaultStyle">
refresh 定时重定向 <meta http-equiv="Refresh" content="10;url=...">
name 把 content 属性关联到一个名称 application-name 应用名称 <meta name="application-name" content="meta-demo">
author 作者名称 <meta name="author" content="234607">
description 页面精简描述 <meta name="description" content="...">
generator 生成页面使用的软件 <meta name="generator" content="vs code">
keywords 关键词 <meta name="keywords" content="html, meta">
viewport 视口设置(移动端) <meta name="viewport" content="width=device-width, ...">
http-equiv="Refresh"
content 概述
<meta http-equiv="Refresh" content="99;url=http://www.baidu.com/#s_kw_wrap" />
99 重新载入页面的时间间隔(秒)
url=http://www.baidu.com/#s_kw_wrap 重定向到指定链接
http-equiv="X-UA-Compatible"
定义浏览器的渲染方式
content 概述
<meta http-equiv="X-UA-Compatible" content="ie=edge,chrome=1">
ie=edge 使用当前 IE 所支持的最高版本标准模式渲染
chrome=1 使用 chrome 内核来渲染
name="viewport"
提供有关视口初始大小的提示，仅供移动设备使用
content 概述
<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">
width=device-width 表示宽度是设备屏幕的宽度
intial-scale=1.0 表示初始的缩放比例
minimum-scale=0.5 表示最小的缩放比例
maximum-scale=2.0 表示最大的缩放比例
user-scalable=yes 表示用户是否可以调整缩放比例
