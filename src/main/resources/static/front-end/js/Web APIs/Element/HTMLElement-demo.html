HTMLElement
属性 概述
HTMLElement.disabled boolean值，表示控件是否被禁用，被禁用的控件不接受任何输入和点击
HTMLElement.form 返回一个父表单元素的引用
HTMLElement.focus() 设置指定元素获取焦点
HTMLElement.blur() 设置指定元素失去焦点
HTMLInput/FormElement.autocomplete 定义输入字段是否应该启用自动完成功能
HTMLFormElement.elements 返回一个 HTMLFormControlsCollection (HTML 4 HTMLCollection)，其中包含FORM的所有控件。
需要注意的是，其中不包括 type="image" 的 input 元素
HTMLFormElement.reset() 重置一个表单内的所有表单控件的值到初始状态
HTMLSelectElement.options 返回当前 select 元素包括的 option 元素对象的集合
HTMLSelectElement.multiple boolean值，表示控件是否支持多选
HTMLSelectElement.remove(index) 根据指定索引删除 option 元素
HTMLSelectElement.selectedIndex 反映了被选中的 <option> 元素的索引值。值为-1时表明没有元素被选中
HTMLTableElement.cellpadding 定义了表格单元的内容和边框之间的空间
HTMLTableElement.cellspacing 定义了两个单元格之间空间的大小
HTMLTableElement.frame 定义表格的外部边框
HTMLTableElement.rules 定义表格的内部边框
HTMLTableElement.rows 返回一个 HTMLCollection，其中包含表的每一行
HTMLTableRowElement.cells 返回一个 HTMLCollection，其中包含行的每一格
HTMLTableElement.createCaption() 返回这个 table 元素的 caption(HTMLTableCaptionElement)。 如果这个 table 没有 caption，这个方法创建并返回 caption
HTMLTableElement.deleteRow(index) 从表中删除一行
HTMLTableElement.insertRow(optional index = -1) 在表中插入一行，并返回新行的引用
HTMLTableRowElement.insertCell(index = -1) 在行中插入一个，并返回新格的引用
