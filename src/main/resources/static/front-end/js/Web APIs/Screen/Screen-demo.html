Screen
属性/方法 概述 示例
height 返回屏幕的高度 screen.height
width 返回屏幕的宽度 screen.width
availHeight 返回屏幕的高度 (除 Windows 任务栏之外) screen.availHeight
availWidth 返回屏幕的宽度 (除 Windows 任务栏之外) screen.availWidth
availTop 返回上屏幕宽度，单屏幕返回 0 screen.availTop
availLeft 返回左屏幕宽度，单屏幕返回 0 screen.availLeft
colorDepth 返回屏幕的颜色深度
根据 CSSOM (CSS对象模型)，为兼容起见，该值总为24 		screen.colorDepth
pixelDepth 返回屏幕的位深度/色彩深度
根据 CSSOM (CSS对象模型)，为兼容起见，该值总为24		 screen.pixelDepth
