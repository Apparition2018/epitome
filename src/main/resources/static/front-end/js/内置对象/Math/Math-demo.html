Math
属性/方法 概述 示例
Math.E 算术常量，欧拉常数，自然对数的底数 约等于 2.718
Math.PI 圆周率 约等于 3.14159
Math.SQRT2 2的平方根 约等于 1.414
Math.abs(x) 返回x的绝对值 Math.abs('-1'); // 1
Math.abs(null); // 0
Math.abs("string"); // NaN
Math.abs(); // NaN
Math.clz32(x) 返回一个32位整数的前导零的数量 Math.clz32(1000) // 22
Math.ceil(x) 返回大于或等于一个给定数字的最小整数 Math.ceil(-0.95); // -0
Math.floor(x) 返回小于或等于一个给定数字的最大整数 Math.floor(-45.05); // -46
Math.round(x) 返回一个数字四舍五入后最接近的整数 Math.round(-20.51) // -21
Math.imul(a, b) 返回两个参数的类C的32位整数乘法运算的运算结果 Math.imul(-1, 8) // -8
Math.imul(0xfffffffe, 5) //-10
Math.max([value1[,value2, ...]]) 返回一组数中的最大值 function getMaxOfArray(numArray) {
    return Math.max.apply(null, numArray);
}
Math.min([value1[,value2, ...]]) 返回零个或更多个数值的最小值
Math.pow(base, exponent) 返回基数（base）的指数（exponent）次幂 Math.pow(2,7) // 128
Math.random() 返回一个浮点, 伪随机数在范围[0，1)
Math.sqrt(x) 返回一个数的平方根 Math.sqrt(2); // 1.414213562373095
Math.trunc(value) 将数字的小数部分去掉，只保留整数部分 Math.trunc("-1.123") // -1
Math.trunc("foo") // NaN
