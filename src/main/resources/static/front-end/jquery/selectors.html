属性 [name] Has Attribute Selector 属性
[name="value"] Attribute Equals Selector 属性等于给定值
[name!="value"] Attribute Not Equal Selector 不存在指定属性，或者指定的属性值不等于给定值 .not('[name="value"]')
[name^="value"] Attribute Starts With Selector 属性以给定字符串开始
[name|="value"] Attribute Contains Prefix Selector 属性值等于给定字符串或以该字符串为前缀（"-"）
[name$="value"] Attribute Ends With Selector 属性以给定值结尾（区分大小写）
[name*="value"] Attribute Contains Selector 属性包含一个给定的子字符串
[name~="value"] Attribute Contains Word Selector 属性用空格分隔的值中包含一个给定值
[name="value"][name2="value2"] Multiple Attribute Selector 匹配所有指定的属性筛选器
基础
过滤		 :eq(index) eq() Selector 在匹配的集合中索引值为index .eq(index)
:lt(index) lt() Selector 在匹配的集合中所有小于给定index .slice(0, index)
:gt(index) gt() Selector 在匹配的集合中所有大于给定index .slice(index)
:first first Selector 第一个匹配，相当于eq(0)，或lt(1) .filter(":first")
:last last Selector 最后一个匹配 .filter(":last")
:odd odd Selector 索引值为奇数，从 0 开始计数 .filter(":odd")
:even even Selector 索引值为偶数，从 0 开始计数 .filter(":even")
:root root Selector 根元素
:focus focus Selector 当前焦点
:lang(language) lang Selector 指定语言
:not(selector) not selector 去除不匹配选择器的
:header header selector 标题 .filter(":header")
:animated animated selector 正在执行动画效果 .filter(":animated")
:target target selector 由文档URI的格式化识别码表示的目标元素 ???
子元素
过滤		 :first-child first-child selector 父级元素下的第一个子
:first-of-type first-of-type selector 相同的元素名称的第一个兄弟
:last-child last-child selector 父级元素下的最后一个子
:last-of-type last-of-type selector 相同的元素名称的最后一个兄弟
:nth-child(index/even/odd/equation) nth-child selector 父级元素下的第n个子
:nth-of-type(index/even/odd/equation) nth-of-type selector 相同的元素名称的第n个兄弟
:nth-last-child(index/even/odd/equation) nth-last-child selector 父级元素下的第n个子，从后算起
:nth-last-of-type(index/even/odd/equation) nth-last-of-type selector 相同的元素名称的第n个兄弟，从后算起
:only-child only-child selector 唯一子元素
:only-of-type only-of-type selector 兄弟中唯一匹配选择器的子元素

:empty empty selector 没有子元素(包括文本节点)
:has(selector) has selector 包含指定选择器 .has(selector/DOMElement)
:parent parent selector 含有子元素或者文本的父级 .filter(":parent")
