# [MyBatis Generator](http://mybatis.org/generator/index.html)
1. [mybatis-generator-maven-plugin](pom.xml)
2. [generatorConfig.xml](src/main/resources/generatorConfig.xml)
3. Running：
    1. [命令提示符](http://mybatis.org/generator/running/runningFromCmdLine.html)：`java -jar mybatis-generator-core-x.x.x.jar -configfile generatorConfig.xml -overwrite`
    2. [Maven](http://mybatis.org/generator/running/runningWithMaven.html)：`Maven → <project_name> → Plugins → mybatis-generator → mybatis-generator:generate`
